CIO / Solv 대시보드 UI 표준화 방안

1. 브라우저 제한
현재 Solv 크롬 접속 불가함. (ie 접속만가능)

2. 해상도차 
CIO(가로:1400) / Solv(가로:1600?)
Solv의 왼쪽 사이드메뉴 구조에 따른 해상도 차이이므로 조정이 힘듬.

3. Main화면 개발시기
완료일 확인 필요(개발계에 붙어작업 할수 없으므로 개발완료후 정확한 작업가능)
디자인 산출물과 퍼블코딩으로 미루어 배경컬러 변경 및 그에 따른 배경과 물린 요소들 css 변경 필요.
현재 개발중인 컬러톤과 차이 다소 있음

4. 예제 : 현재 세부화면 UI에 style.css 조정만으로 가능한 범위 적용예

최대한 구조 변경 없이 css 수정만으로 look & feel 맞추고자함

1) Logo
CIO와 동일한 폰트로 맞춤

2) GUI 
사용자 권한에 따라 메뉴가 늘경우 해상도에 따라 메뉴 위치 조정 불가할수 있음.(현재 Solv는 상하 2단 구조)
조정 불가할경우 GUI 통일성 떨어짐. 권한별 메뉴 구조 확인 필요.

3) CIO 링크
링크 위치 협의 필요.

4) Icon Menu
CIO 스타일로 톤조정.

5) Side Menu
CIO 주요 컬러로 변경.

6) Body Layout
구조상 배경 컬러 변경시 페이지의 Html 마크업 조정이 필요할수 있고 CIO와 같은 카드 UI 적용시 추가 여백등이 조정 되면 레이아웃 틀어짐이 일어 날수 있음. 
현재 컬러 (화이트) 유지필요.

7) Title
CIO와 스타일 통일가능

8) Path & Help
CIO와 스타일 통일가능

9) Tab
CIO 스타일로 조정 (예외 레이아웃이 있어 작업시 변경될수 있음)

0) Search
셀렉트박스와 버튼 CIO 스타일로 조정

a) 주요강조컬러 CIO 블루로 변경

b) 리스트
CIO와 스타일로 컬러변경

5. 작업방안
style.css 수정본 초안 전달후
퍼블리셔가 개발URL로 확인하여 css 수정하고 다시 전달하는 방식
